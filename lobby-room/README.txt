Lobby Room is where tourists have access to the system for creating a
visit and the corresponding group of participants. The main function of a lobby 
is to create a group for a visit, and to enable a tourist to join an existing 
group. 

Joining a group/visit means going to have access to the corresponding visit 
group communication system. 

Implemented using RabbitMQ broker. 

To execute and test, launch the following command:
$ mvn clean install

To run in a console using shell commands the different entities with
the same scenarios as the one programmed in the JUnit tests, execute the
following scripts:
$ ./run.sh
